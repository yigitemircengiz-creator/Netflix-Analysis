# Netflix Global Content Trends - Detailed EDA 

This project focuses on performing an **Exploratory Data Analysis (EDA)** on the Netflix Movies and TV Shows dataset to uncover insights about global streaming trends.

## Project Overview
As a Computer Engineering student, I developed this project to practice advanced data manipulation techniques using **Python** and **Pandas**. The goal was to clean "messy" data and visualize the distribution of content across different regions and genres.

## Tech Stack & Skills
- **Language:** Python
- **Libraries:** Pandas, Matplotlib, Seaborn
- **Key Techniques:** - Data Cleaning (Handling missing values with `fillna` and `dropna`)
  - Data Transformation (Using `explode` for multi-genre strings)
  - Time-series Analysis (Converting strings to `datetime` objects)
  - Data Visualization (Pie charts, Bar charts, and Line plots)

## Key Insights
- **Content Mix:** Movies represent approximately **70%** of the total content, while TV Shows make up the remaining **30%**.
- **Top Players:** The United States and India are the leading content producers.
- **Genre Dominance:** International Movies and Dramas are the most frequent genres on the platform.
- **Localized Growth:** Analyzed the specific growth and distribution of Turkish productions over the years.

## How to Run
1. Clone this repository.
2. Ensure you have `pandas`, `matplotlib`, and `seaborn` installed.
3. Run the `.ipynb` notebook in a Jupyter environment or Google Colab.
